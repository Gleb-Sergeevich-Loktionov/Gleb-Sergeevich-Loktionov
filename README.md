### 🧠 About · Обо мне

**Food-production engineer turned AI builder.** I design agent orchestration and automation the way I once designed manufacturing lines — **deterministic where it must be, adaptive where it pays off.** My bar isn't a flashy demo; it's a system that keeps running when nobody is watching.

> Я инженер пищевого производства, ушедший в AI. Проектирую оркестрацию агентов и автоматизацию так же, как раньше проектировал производственные линии: детерминированно там, где это критично, и адаптивно там, где это даёт выигрыш. Моя планка — не красивое демо, а система, которая работает, когда на неё никто не смотрит.

### 🚀 What I build

- 🤖 **Multi-agent systems** — Solver / Critic / Judge loops, agent orchestration, LLM tooling
- 🔌 **Automation & integrations** — async Python pipelines and API plumbing that run unattended
- 💬 **Telegram bots & tooling** — `aiogram`-based interfaces and workflow bots
- 🧩 **Local + cloud LLMs** — Claude API for reasoning, Ollama for on-prem / private inference
- 📦 **Containerized delivery** — reproducible runs with Docker
